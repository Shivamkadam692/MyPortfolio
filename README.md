## Getting Started

1. Clone or download this repository to your local machine
2. Ensure all assets (`styles.css`, `script.js`, `PHOTO.jpg`, `ShivamPrakashKadamResume.pdf`) are placed alongside `index.html`
3. Open `index.html` in any modern web browser to view the portfolio
4. Toggle between light/dark themes using the theme switcher in the navigation bar

## Customization

- Update the hero and about sections in `index.html` to reflect new achievements or roles
- Modify the `skills` grid to add or reorder skill items; icons leverage [Font Awesome 6](https://fontawesome.com/icons)
- Add new projects by duplicating a `project-card` within the Projects section and updating the details
- Adjust styling in `styles.css` or interactions in `script.js` as needed
- Customize color schemes and animations in the CSS variables section
